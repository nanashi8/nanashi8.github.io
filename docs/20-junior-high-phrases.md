# 20. 中学熟語データセット仕様書

## 📚 概要

中学英語で習得すべき重要熟語500表現のデータセットです。熟語・慣用句・会話表現を含み、実用的なコミュニケーション能力の向上を目指します。

---

## 🎯 データ仕様

### 1. ファイル形式

- **フォーマット**: CSV（7列）
- **エンコーディング**: UTF-8
- **総表現数**: 約500表現
- **ファイル**: `public/data/batch*-phrases-*.csv`

### 2. データ例

```csv
look for,ルック フォー,探す,look(見る) + for(〜を),look at(見る),動作,初級
be good at,ビー グッド アット,得意である,be動詞 + 形容詞 + 前置詞,be interested in,性質,初級
```

### 3. 特徴

- **スペース区切り**: 複数単語から成る表現
- **isPhraseOnly**: trueフラグで熟語判定
- **会話表現**: How are you? など

### 4. カテゴリー

- 動詞句
- 形容詞句
- 副詞句
- 会話表現
- 前置詞句
