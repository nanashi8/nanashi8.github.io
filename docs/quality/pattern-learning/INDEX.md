# パターン学習教材（全パッセージ）

## 目的
全パッセージの /分割・()分割・直訳・一文訳・語句分割を**誤りゼロ**で確定し、教材品質を保証する。

## 参照ルール
- 品質ルールは [docs/guidelines/passage/PASSAGE_PHRASE_JSON_CREATION_GUIDE.md](../guidelines/passage/PASSAGE_PHRASE_JSON_CREATION_GUIDE.md) を正とする。
- 原文の正本は public/data/passages/1_passages-original を正とする。

## 作業フロー
1. 原文確認
2. /分割（句・節境界）
3. ()分割（<>句、()節）
4. 直訳（フレーズ訳：句・節ごと）
5. 一文訳（英文・和訳を交互）
6. 語句分割（+ ボタン単位）
7. レビュー → ルール固定
