##### 3-1. 助詞（新規20件）

**格助詞**
- が（主格・連体格）

- へ（方向）
- と（引用・並列・比較）
- より（起点・比較）

**接続助詞**
- ば（順接条件・恒時条件）
- ど・ども（逆接）
- が（逆接）
- に（原因理由）
- を（逆接）
- て（接続）
- で（打消接続）

**副助詞**
- だに（最小限・類推）
- すら（類推）
- さへ（添加）
- のみ（限定）

**係助詞**
- ぞ（強調→連体形）

**終助詞**
- な（禁止）

## 4. データ品質基準

### 各項目の必須フィールド

#### CSV構造
```
語句,読み,意味,解説,関連事項,時代背景,種別
```

### 品質チェック項目

#### ✅ 必須要件
1. **語句**: 古文での表記（歴史的仮名遣い）
2. **読み**: 現代仮名遣いでの読み
3. **意味**: 簡潔な現代語訳（3-5語程度）
4. **解説**: 詳細な説明（100-200文字）
   - 語の成り立ちや由来
   - 使用例（「〜の形で」）
   - 現代語との違い
   - 類義語・対義語
5. **関連事項**: パイプ区切りのタグ（3-6個）
6. **時代背景**: 時代区分（平安時代、鎌倉時代など）
7. **種別**: カテゴリー分類

#### 🎯 推奨要件
- 例文の追加（有名作品からの引用）
- 類似問題への誘導
- 暗記のコツ
