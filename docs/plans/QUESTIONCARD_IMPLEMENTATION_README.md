## 🚨 よくある問題と解決策

### Q1: TypeScriptエラーが出る
**A:** CustomWord型の必須フィールドを確認。`src/types/customQuestions.ts`を参照。

### Q2: ボタンが表示されない
**A:** 親コンポーネント(App.tsx等)からpropsが正しく渡されているか確認。

### Q3: レイアウトが崩れる
**A:** クイックスタートガイドのトラブルシューティングを参照。

### Q4: クリックしても反応しない
**A:** イベントハンドラーとpropsの確認。ブラウザコンソールでエラーをチェック。
