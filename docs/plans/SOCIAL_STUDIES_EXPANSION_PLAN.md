# 社会科問題拡充計画

## 📋 概要

**目的**: 東京書籍『新編 新しい社会』準拠の中学社会科暗記教材を、高校受験頻出分野を中心に拡充する

**対象教材**: 東京書籍『新編 新しい社会』シリーズ（令和3年度版）
- 歴史的分野
- 地理的分野
- 公民的分野

**現在の規模**: 108語句（歴史39/地理30/公民29/その他10）  
**目標規模**: 1,500語句（3年間の学習範囲をカバー）

---

## 🎯 拡充方針

### 1. 東京書籍準拠（最優先）

東京書籍『新編 新しい社会』の章立てと重要語句に準拠:

**公式資料**:
- 年間指導計画: https://ten.tokyo-shoseki.co.jp/text/chu/list/keikaku/#shakai
- 教師用指導書準拠の重要語句リスト
- 定期テスト・実力テストでの頻出語句

**実装方針**:
- 教科書の章ごとに重要語句を抽出
- 教科書の太字語句を優先的に収録
- 各単元の「まとめ」「確認」ページの語句を網羅

### 2. 高校受験頻出分野（第2優先）

都道府県別公立高校入試過去問（過去5年）の出題頻度分析に基づく:

#### 歴史分野（目標600語句）
- **古代** (100語句): 縄文・弥生・古墳・飛鳥・奈良時代
  - 頻出: 邪馬台国、卑弥呼、大化の改新、聖徳太子、天平文化
- **中世** (100語句): 平安・鎌倉・室町時代
  - 頻出: 平氏政権、源平合戦、鎌倉幕府、元寇、室町文化
- **近世** (150語句): 安土桃山・江戸時代
  - 頻出: 織田信長、豊臣秀吉、徳川家康、鎖国、享保の改革、寛政の改革、天保の改革
- **近代** (150語句): 明治・大正時代
  - 頻出: 明治維新、五箇条の御誓文、版籍奉還、廃藩置県、自由民権運動、大日本帝国憲法、日清戦争、日露戦争、第一次世界大戦
- **現代** (100語句): 昭和・平成時代
  - 頻出: 太平洋戦争、ポツダム宣言、日本国憲法、冷戦、高度経済成長、バブル経済

#### 地理分野（目標450語句）
- **日本地理** (200語句):
  - 頻出: 地形（山脈・平野・河川）、気候、農業（稲作・畑作・畜産）、工業（工業地帯・工業地域）
  - 都道府県別特色（47都道府県の主要都市・特産品・産業）
- **世界地理** (150語句):
  - 頻出: 六大陸・三大洋、気候帯、主要国の位置・首都・特色
  - アジア（中国・韓国・インド等）、ヨーロッパ（EU加盟国）、アメリカ、アフリカ
- **産業・環境** (100語句):
  - 頻出: 第一次産業（農業・林業・水産業）、第二次産業（工業・鉱業）、第三次産業（サービス業）
  - 環境問題（地球温暖化・酸性雨・砂漠化）、エネルギー問題（化石燃料・再生可能エネルギー）

#### 公民分野（目標450語句）
- **政治** (200語句):
  - 頻出: 日本国憲法（三大原則・基本的人権・国民主権・平和主義）
  - 国会・内閣・裁判所の仕組み、三権分立、選挙制度、地方自治
- **経済** (150語句):
  - 頻出: 市場経済（需要・供給・価格）、企業（株式会社・中小企業）
  - 金融（日本銀行・金融政策）、財政（税金・社会保障）、消費者問題
- **国際** (100語句):
  - 頻出: 国際連合、国際協力（ODA・NGO・NPO）
  - 地球規模の課題（環境・人権・貧困・紛争）、グローバル化

## 📝 データ作成基準

### CSV形式（8列）

```csv
語句,読み,意味,詳細解説,関連事項,関連分野,種別,source
```

#### フィールド定義

1. **語句**: 覚えるべき用語（20文字以内）
2. **読み**: ひらがな表記（必須、40文字以内）
3. **意味**: 一言説明（30文字以内）
4. **詳細解説**: 詳しい説明（200文字以内、背景・原因・結果を含む）
5. **関連事項**: パイプ区切り（例: `関ヶ原の戦い|大坂の陣|幕藩体制`）
6. **関連分野**: 分野名（例: `歴史-近世`, `地理-日本`, `公民-政治`）
7. **種別**: `人物名`, `地名`, `出来事`, `制度概念`, `その他`
8. **source**: `history`, `geography`, `civics`, `junior`（統合用は`junior`）

### 品質基準

#### 必須要件
- ✅ 教科書準拠: 東京書籍『新編 新しい社会』の章立てに沿う
- ✅ 入試頻出: 過去5年の公立高校入試で3回以上出題
- ✅ 詳細解説: 背景・原因・結果を含む200文字以内の説明
- ✅ 関連事項: 最低2つ以上の関連語句を明記

#### 禁止事項
- ❌ 教科書に記載のない独自見解
- ❌ 政治的・宗教的に偏った表現
- ❌ 時代遅れの情報（統計データは最新年度を使用）

### 品質チェック

作成後は以下のスクリプトで検証:

```bash
npm run validate:social-studies local-data-packs/社会科問題.csv
```
