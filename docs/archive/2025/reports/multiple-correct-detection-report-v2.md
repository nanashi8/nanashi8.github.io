### g3-u4-fib-029 - 関係代名詞 (grammar_grade3_unit4.json)

**日本語**: 私はどちらが正しいか分かりません。

**英文**: I don't know ____ is correct.

**正解**: `which`

**選択肢**: `which`, `what`, `who`

**検出理由**:
- 関係代名詞が2個: which, who

【関係代名詞の使い分け】
- which: 物・動物を先行詞とする
- who: 人を先行詞とする主格・目的格

⚠️ 先行詞が不明瞭 → 文脈により複数正解の可能性


**推奨対応**: 先行詞を明確にするか、複数正解として扱う

**現在の解説**:
```
間接疑問文です。「どちらが」という選択を尋ねるのでwhichを使います。

正解は「which」です。

【日本語】
私はどちらが正しいか分かりません。

【文法ポイント】
間接疑問文、現在分詞・過去分詞の修飾...
```
