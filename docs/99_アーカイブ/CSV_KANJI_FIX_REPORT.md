## 📊 修正方針

### 基本ルール
1. **動物名**: カタカナ表記（外来語として定着しているもの、小動物）
2. **基本的な名詞**: 漢字表記（小学校で習う漢字）
3. **一般的な動詞・形容詞**: 漢字表記
4. **和訳列**: 原則として変更しない（学習者のため）
5. **解説文**: 適切な表記に統一

### 例外
- 「りんご」は「リンゴ」（カタカナが一般的）
- 「ウサギ」「カエル」などの小動物はカタカナ
- 「牛」「馬」「犬」「猫」「鳥」「魚」などは漢字（常用漢字）
