# 中学生・高校受験英熟語集作成仕様書

## 📌 概要

中学生が学習する高校受験レベルの英熟語（イディオム・句動詞・コロケーション）集を作成するための詳細仕様。和訳クイズとスペルクイズの両方で使用可能な、CSV形式のデータを生成します。

**対象レベル:** 中学生(高校受験対応:初級・中級・上級)  
**目標熟語数:** 1,000〜1,200熟語（初級400、中級400、上級200〜400）  
**フォーマット:** CSV(UTF-8 エンコーディング)  
**出力ファイル:** `public/data/junior-high-entrance-phrases.csv`（または統合ファイル）  
**既存単語データ:** `junior-high-entrance-words.csv` (3,600語)

---

## 🎯 熟語の定義と範囲

### 熟語の種類

本仕様書では以下の3種類の熟語を対象とします：

#### 1. **句動詞 (Phrasal Verbs)**
動詞 + 前置詞/副詞の組み合わせで、元の動詞とは異なる意味を持つ表現。

**例:**
- `look after` (世話をする) ← look(見る) + after(後に)
- `give up` (あきらめる) ← give(与える) + up(上に)
- `turn on` (スイッチを入れる) ← turn(回す) + on(上に)

**特徴:**
- 高校受験で最も重要
- 動詞の基本的な意味から推測困難
- 目標: 400〜500件

#### 2. **慣用句 (Idioms)**
複数の単語が結びついて、字面からは推測できない慣用的な意味を持つ表現。

**例:**
- `a piece of cake` (とても簡単なこと)
- `break the ice` (打ち解ける)
- `cost an arm and a leg` (とても高価である)

**特徴:**
- 文化的背景を含む
- 比喩的表現が多い
- 目標: 300〜400件

#### 3. **重要コロケーション (Collocations)**
特定の単語同士が習慣的に組み合わされる表現。

**例:**
- `make a decision` (決定を下す) ※ do a decision とは言わない
- `take a bath` (風呂に入る)
- `pay attention` (注意を払う)

**特徴:**
- 自然な英語表現に必須
- 単語知識だけでは不十分
- 目標: 300〜400件

## 📊 データ構造

### CSV形式（既存7列との互換性維持）

```csv
語句,読み,意味,語源等解説,関連語,関連分野,難易度
```

### フィールド詳細（熟語特有の記述方法）

#### 1. 語句（phrase）
- **型:** string
- **説明:** 英熟語・イディオム
- **例:** `look after`, `a piece of cake`, `make a decision`
- **ルール:**
  - 小文字で記述
  - 単語間はスペースで区切る
  - 冠詞（a/an/the）を含む場合はそのまま記載
  - 句動詞は動詞原形で記載（例: `look after`, not `looks after`）

#### 2. 読み（reading）
- **型:** string
- **説明:** アクセント記号を正確に表したカタカナ読み
- **例:** `ルック アフター`, `ア ピース オブ ケイク`
- **ルール:**
  - 各単語の主強勢にアクセント記号（́）を付与
  - 単語間は半角スペースで区切る
  - 機能語（前置詞・冠詞）は弱形で発音される場合も記載
- **例:**
  - `look after` → `ルッ́ク ア́フター`
  - `a piece of cake` → `ア ピ́ース オヴ ケ́イク`
  - `make a decision` → `メ́イク ア ディシ́ジョン`

#### 3. 意味（meaning）
- **型:** string
- **説明:** 日本語の意味
- **例:** `〜の世話をする`, `とても簡単なこと`, `決定を下す`
- **ルール:**
  - 中学生が理解できる平易な日本語
  - 動詞句の場合は「〜する」形式
  - 名詞句の場合は名詞形
  - 複数の意味がある場合は「・」で区切る（最重要義を先頭に）

#### 4. 語源等解説（explanation）
- **型:** string
- **説明:** 熟語の成り立ち、語源、使い方のヒント
- **ルール:**
  - 構成要素の説明（句動詞の場合）
  - 比喩の由来（慣用句の場合）
  - 使用場面や注意点
  - 関連する句動詞・イディオムへの言及
- **例:**
  ```
  "look"(見る)と"after"(後に)の組み合わせ。「後ろから見守る」というイメージから「世話をする」の意味に。同様の句動詞: take care of(テ́イクケ́アオブ): 世話をする
  ```
  ```
  直訳は「ケーキ一切れ」。ケーキを食べるのは簡単なことから、「とても簡単」という意味の慣用句になった。類似表現: easy as pie(イ́ージーアズパ́イ): とても簡単
  ```

#### 5. 関連語（relatedWords）
- **型:** string（ダブルクオートで囲む）
- **説明:** 類似表現・反意表現・関連熟語のリスト
- **形式:** `"熟語1(読み1): 意味1, 熟語2(読み2): 意味2, ..."`
- **ルール:**
  - 必ずダブルクオートで全体を囲む
  - 各関連語は「熟語(読み): 意味」のトリプル形式
  - 2〜4個程度を推奨
  - 類義語を優先、次に反意語、関連表現の順
- **例:**
  ```
  "take care of(テ́イクケ́アオブ): 世話をする, watch over(ウォ́ッチオーヴァー): 見守る"
  ```

#### 6. 関連分野（category）
- **型:** string
- **説明:** 熟語が属する分野・カテゴリ
- **ルール:**
  - 既存の10カテゴリから選択
  - 熟語の意味内容に基づいて分類（構成要素ではない）
  
**既存10カテゴリ:**
```
1. 言語基本 (基本動詞・形容詞・副詞・前置詞・接続詞・代名詞など)
2. 学校・学習 (教育・授業・科目・学問・知識・読み書きなど)
3. 日常生活 (家庭・住居・日課・買い物・家事など)
4. 人・社会 (人間関係・感情・性格・職業・ビジネス・経済など)
5. 自然・環境 (動物・植物・天候・地理・環境問題など)
6. 食・健康 (食べ物・料理・身体・医療・衛生など)
7. 運動・娯楽 (スポーツ・趣味・芸術・音楽・行事など)
8. 場所・移動 (交通・旅行・方向・位置・建物など)
9. 時間・数量 (時制・数学・測定・数・量など)
10. 科学・技術 (テクノロジー・実験・道具・機械・通信など)
```

**分類例:**
- `look after` → 日常生活（世話をする行為）
- `give up` → 人・社会（感情・意志）
- `a piece of cake` → 言語基本（程度表現）
- `break the ice` → 人・社会（人間関係）

#### 7. 難易度（difficulty）
- **型:** string
- **説明:** 高校受験における難易度レベル
- **値:** `初級`, `中級`, `上級` のいずれか（厳密一致）
- **目安:**
  - **初級:** 中学1〜2年レベル、超頻出句動詞・基本イディオム（約400熟語）
    - 例: `look at`, `get up`, `go to`, `come back`
  - **中級:** 中学3年〜高校受験標準レベル（約400熟語）
    - 例: `look forward to`, `take care of`, `break down`
  - **上級:** 高校受験難関校レベル、応用熟語（約200〜400熟語）
    - 例: `do away with`, `put up with`, `come down with`

## 📝 作成ガイドライン

### 熟語選定基準

#### 1. **頻出度**
- 高校受験過去問での出現頻度
- 英検3級〜準2級レベル
- 複数の参考書・単語帳に掲載

#### 2. **重要性**
- 日常会話・読解で頻繁に使用
- 置き換えが困難（他の表現では代替不可）
- 中学生の語彙力拡張に効果的

#### 3. **学習効率**
- 同じ動詞の句動詞をグループ化（例: get up, get on, get off）
- 体系的に学習可能
- 派生的に覚えやすい

### 熟語タイプ別の配分目安

| タイプ | 目標数 | 割合 |
|--------|--------|------|
| 句動詞 | 500件 | 45% |
| 慣用句 | 350件 | 32% |
| コロケーション | 250件 | 23% |
| **合計** | **1,100件** | **100%** |

### 作成プロセス

#### ステップ1: 熟語リスト作成（難易度別）
1. 高校受験用熟語集・過去問から頻出表現を抽出
2. 動詞別・テーマ別に整理
3. 難易度を仮割り当て

#### ステップ2: データ生成（100熟語単位）
1. 熟語・意味を確定
2. **読みを辞書等で確認し、正確なアクセント記号を付与**
3. 語源・成り立ちを調査・記述
4. 関連熟語を2〜4個選定（読み・意味を付記）
5. 適切なカテゴリを選択
6. 難易度を最終確認

#### ステップ3: CSV追記
1. 末尾追記方式でCSVに追加
2. カンマ数、ダブルクオートの整合性をチェック
3. UTF-8エンコーディングを確認

#### ステップ4: 検証
1. CSV構文エラーがないか確認
2. 読みのアクセント記号の抜け漏れをチェック
3. カテゴリが一覧にあるか確認
4. 難易度が「初級・中級・上級」のいずれかか確認
5. 重複チェック（既存単語・熟語との重複）

## 🔧 実装詳細

### データファイル構成（推奨）

**オプションA: 統合ファイル型（推奨）**
```
public/data/junior-high-entrance-vocabulary.csv
├── 単語セクション (3,600件)
└── 熟語セクション (1,100件)
```
- メリット: シンプル、単一ファイル管理
- デメリット: ファイルサイズ増大（4,700行程度）

**注:** 単語と熟語を区別するため、将来的に`type`フィールド追加を検討

### バッチ処理方式

**推奨:** 100熟語単位でのサブバッチ処理

```
100熟語生成 → CSV追記 → 検証 → 進捗報告 → 次の100熟語...
```

**処理フロー:**
```
初級100 → 初級100 → 初級100 → 初級100 (計400)
→ 中級100 → 中級100 → 中級100 → 中級100 (計400)
→ 上級100 → 上級100 → 上級100 (計300)
```

### CSV追記時の注意点

1. **スペース:**
   - 熟語内のスペースはそのまま記載（例: `look after`）
   - 読み方のスペースも半角で記載（例: `ルッ́ク ア́フター`）

2. **ダブルクオート:**
   - `relatedWords`フィールドは必ずダブルクオートで囲む
   - 熟語内にカンマが含まれる場合もダブルクオートで囲む

3. **カンマ:**
   - `meaning`フィールドでは複数意味の区切りに「・」を使用
   - カンマは列区切り専用

4. **エンコーディング:**
   - UTF-8（BOM なし）

## ✅ 品質チェックリスト

各バッチ完了時に以下を確認:

### データ品質
- [ ] 全ての熟語に**アクセント記号付き**読みが付いているか
- [ ] スペースの位置が正確か（熟語内、読み方内）
- [ ] 語源等解説に成り立ち・使い方が含まれているか
- [ ] 関連語が`"熟語(読み): 意味"`形式で記述されているか
- [ ] カテゴリが指定リストに存在するか
- [ ] 難易度が「初級・中級・上級」のいずれかか

### 技術品質
- [ ] CSV構文エラー（カンマ数、クオート不整合）がないか
- [ ] UTF-8エンコーディングか
- [ ] 重複熟語がないか（熟語間、単語との重複）
- [ ] 行末に余計なスペースや改行がないか

### 教育品質
- [ ] 中学生が理解できる説明か
- [ ] 高校受験で実際に出題される熟語か
- [ ] 難易度設定が適切か
- [ ] 関連語が学習に役立つか
