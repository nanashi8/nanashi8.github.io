# 01. プロジェクト概要

## 📌 基本情報

**プロジェクト名**: 中学受験英語学習アプリ  
**バージョン**: 3.1.0  
**最終更新**: 2025年11月25日  
**ライセンス**: MIT  
**デプロイURL**: https://nanashi8.github.io/

## 🎯 目的

中学受験を控えた小学生と中学生が効率的に英語学習できるインタラクティブなWebアプリケーション。
5つの学習モード（和訳・スペル・長文読解・文法問題×3種）と7つのAI機能により、個別最適化された学習体験を提供。

## 👥 ターゲットユーザー

- **主要**: 中学受験を目指す小学5〜6年生
- **副次**: 中学1〜3年生（NEW HORIZON準拠の文法問題対応）
- **保護者**: 学習進捗を把握したい保護者

## 🎨 主要機能

### 学習モード（5種類）

1. **和訳クイズ**
   - 英単語/フレーズから日本語訳を選ぶ3択クイズ
   - 難易度フィルター（初級・中級・上級）
   - 単語/フレーズフィルター
   - 音声読み上げ機能

2. **スペルクイズ**
   - 日本語訳から英語スペルを入力
   - 虫食い形式とフル入力形式
   - リアルタイム入力検証
   - ヒント機能

3. **長文読解**
   - レベル別の長文パッセージ（21本）
   - Beginner: 5本 / Intermediate: 8本 / Advanced: 8本
   - セグメント単位の読解
   - 未学習単語の自動抽出
   - 音声読み上げ機能

4. **文法問題（NEW HORIZON準拠）**
   - **文並び替え**: 1年170問・2年170問・3年170問（計510問）
   - **動詞変化**: 1年153問・2年170問・3年170問（計493問）
   - **穴埋め**: 1年160問・2年160問・3年160問（計480問）
   - 学年・単元フィルター対応
   - 東京書籍『NEW HORIZON English Course』完全準拠
   - **総問題数**: 1,483問

### AI機能（7種類）

1. **認知負荷AI** - 疲労度を測定し難易度を自動調整
2. **エラー予測AI** - 過去の誤答パターンから苦手単語を予測
3. **文脈学習AI** - 関連単語をグループ化して出題
4. **学習スタイルAI** - 時間帯・正答率から学習傾向を分析
5. **ゲーミフィケーションAI** - XP・レベル・実績でモチベーション維持
6. **学習曲線AI** - 忘却曲線に基づく復習タイミング予測
7. **レーダーチャートAI** - 6カテゴリー（動詞・名詞等）の習熟度分析

### 統計・分析機能

- **スコアボード**: 5つのタブで多角的に分析
  - 統計: 正答率・学習時間・連続正解数
  - 定着度分析: 完全定着・学習中・要復習の内訳
  - 目標設定: デイリー・ウィークリー目標と進捗
  - 学習プラン: 今日・今週・今月の推奨学習量
  - 履歴: 日別・週別の学習履歴グラフ

- **レーダーチャート**: 品詞別習熟度の可視化
- **定着度レート**: 単語ごとの定着度(0-100%)
