# 01. プロジェクト概要

## 📌 基本情報

**プロジェクト名**: 中学受験英語学習アプリ  
**バージョン**: 3.0.0  
**最終更新**: 2025年11月22日  
**ライセンス**: MIT  
**デプロイURL**: https://nanashi8.github.io/

## 🎯 目的

中学受験を控えた小学生が効率的に英語学習できるインタラクティブなWebアプリケーション。
3つの学習モード（和訳・スペル・長文読解）と7つのAI機能により、個別最適化された学習体験を提供。

## 👥 ターゲットユーザー

- **主要**: 中学受験を目指す小学5〜6年生
- **副次**: 英語学習を始めたばかりの中学1年生
- **保護者**: 学習進捗を把握したい保護者

## 🎨 主要機能

### 学習モード（3種類）

1. **和訳クイズ**
   - 英単語/フレーズから日本語訳を選ぶ3択クイズ
   - 難易度フィルター（初級・中級・上級）
   - 単語/フレーズフィルター
   - 音声読み上げ機能

2. **スペルクイズ**
   - 日本語訳から英語スペルを入力
   - 虫食い形式とフル入力形式
   - リアルタイム入力検証
   - ヒント機能

3. **長文読解**
   - レベル別の長文パッセージ（10本）
   - セグメント単位の読解
   - 未学習単語の自動抽出
   - 音声読み上げ機能

### AI機能（7種類）

1. **認知負荷AI** - 疲労度を測定し難易度を自動調整
2. **エラー予測AI** - 過去の誤答パターンから苦手単語を予測
3. **文脈学習AI** - 関連単語をグループ化して出題
4. **学習スタイルAI** - 時間帯・正答率から学習傾向を分析
5. **ゲーミフィケーションAI** - XP・レベル・実績でモチベーション維持
6. **学習曲線AI** - 忘却曲線に基づく復習タイミング予測
7. **レーダーチャートAI** - 6カテゴリー（動詞・名詞等）の習熟度分析

### 統計・分析機能

- **スコアボード**: 5つのタブで多角的に分析
  - 統計: 正答率・学習時間・連続正解数
  - 定着度分析: 完全定着・学習中・要復習の内訳
  - 目標設定: デイリー・ウィークリー目標と進捗
  - 学習プラン: 今日・今週・今月の推奨学習量
  - 履歴: 日別・週別の学習履歴グラフ

- **レーダーチャート**: 品詞別習熟度の可視化
- **定着度レート**: 単語ごとの定着度(0-100%)

## 🛠️ 技術スタック

### フロントエンド
- **React 18.2** + **TypeScript 5.2**
- **Vite 5.4** - ビルドツール
- **CSS3** - カスタムスタイリング（CSS変数対応）

### ストレージ
- **IndexedDB** - メイン学習データ（10MB+対応）
- **LocalStorage** - 設定データ（軽量データ）

### デプロイ
- **GitHub Pages** - 静的サイトホスティング
- **gh-pages** - 自動デプロイツール

### 開発ツール
- **ESLint** - コード品質チェック
- **Git** - バージョン管理
- **VS Code** - 推奨エディタ

## 📊 データ規模

- **単語数**: 約800語（中学受験頻出語彙）
- **フレーズ数**: 約500フレーズ
- **長文パッセージ**: 10本（初級3本、中級5本、上級2本）
- **総単語カバレッジ**: 約1,500ユニーク単語

## 🎨 UI/UXの特徴

1. **ダークモード対応** - 目に優しい暗めの配色
2. **レスポンシブデザイン** - スマホ・タブレット・PCに対応
3. **タッチ最適化** - iOSでの快適な操作性
4. **固定レイアウト** - 画面スクロールを最小化
5. **リアルタイムフィードバック** - 即座に正誤判定

## 🔒 プライバシー

- **データ保存**: すべてブラウザ内（サーバー送信なし）
- **個人情報**: 収集・保存一切なし
- **オフライン動作**: 一度読み込めばオフラインで利用可能
