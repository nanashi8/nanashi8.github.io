# 解説テンプレート集

**作成日**: 2025年12月15日  
**対象**: 文法問題解説の標準化

---

## 📋 テンプレート使用ガイド

### 基本原則

1. **明確性**: 中学生が理解できる平易な表現
2. **簡潔性**: 200-400文字を目安
3. **構造化**: セクション分けで読みやすく
4. **具体性**: 抽象的な説明より具体例を重視

### 変数の使い方

- `{和訳}`: 問題文の日本語訳
- `{文法事項}`: 該当する文法ポイント
- `{正解}`: 正しい答え
- `{誤答例}`: よくある間違い

## 1. sentenceOrdering（並び替え問題）

### 1-1. 基本語順パターン

```markdown
「{和訳}」です。

【文法ポイント】
{文法事項の説明（時制、文型など）}

【語順ルール】
{主語} + {動詞} + {目的語/補語} + {修飾語句}
※{語順の理由や原則}

【よくある間違い】
× {間違った語順}
○ {正しい語順}
```

**適用例:**
```markdown
「私は毎日学校へ行きます」です。

【文法ポイント】
一般動詞の現在形（主語Iは原形go）

【語順ルール】
I + go + to school + every day
※「場所(to school)」→「頻度(every day)」の順

【よくある間違い】
× I every day go to school
○ I go to school every day
```

### 1-2. 疑問文パターン

```markdown
「{和訳}」という疑問文です。

【文法ポイント】
疑問文の語順: {Do/Does/Did/疑問詞} + 主語 + 動詞原形 ...?

【語順ルール】
{疑問詞の位置と理由}

【注意点】
・動詞は必ず原形に戻す
・{その他の注意事項}
```

**適用例:**
```markdown
「あなたは何が欲しいですか？」という疑問文です。

【文法ポイント】
疑問詞whatが主語の後に来る: What + do + you + want?

【語順ルール】
疑問詞 + 助動詞 + 主語 + 動詞原形

【注意点】
・wantは原形のまま（wants×）
・疑問文では必ずdoを使用
```

### 1-3. 副詞位置パターン

```markdown
「{和訳}」です。

【文法ポイント】
{副詞の種類}の位置ルール

【語順ルール】
{基本語順} + {副詞の位置}
※{副詞位置の原則}

【副詞の種類別】
・時を表す副詞: 通常は文末
・場所を表す副詞: 時の前
・頻度の副詞: 一般動詞の前、be動詞の後
```

**適用例:**
```markdown
「私たちは昨日公園で走りました」です。

【文法ポイント】
過去形 + 場所・時の副詞句

【語順ルール】
We + ran + in the park + yesterday
※場所(in the park) → 時(yesterday)の順

【副詞の種類別】
・場所の副詞句: in the park（動詞の直後）
・時の副詞: yesterday（文末）
```

## 2. fillInBlank（穴埋め問題）

### 2-1. 動詞形パターン

```markdown
{正解形} が正解です。

【理由】
{主語の人称・時制による変化の説明}

【文法ポイント】
{該当する時制や文型}

【よくある間違い】
× {誤答}: {なぜ間違いか}
```

**適用例:**
```markdown
studies が正解です。

【理由】
主語がShe（三人称単数）で、現在形なので動詞に-sをつけます。

【文法ポイント】
三人称単数現在形: study → studies（子音+yは-iesに変化）

【よくある間違い】
× study: 三人称単数では-sが必要
× studing: 現在進行形ではない
```

### 2-2. 前置詞パターン

```markdown
{前置詞} が正解です。

【理由】
{前置詞の意味・用法}

【使い分け】
{類似前置詞との違い}

【熟語】
{関連する熟語表現}
```

**適用例:**
```markdown
at が正解です。

【理由】
「〜時に」という時刻を表す場合はatを使います。

【使い分け】
・at: 時刻（at 3 o'clock）
・on: 曜日・日付（on Monday）
・in: 月・年・季節（in April）

【熟語】
at night（夜に）、at noon（正午に）
```

### 2-3. 助動詞パターン

```markdown
{助動詞} が正解です。

【理由】
{助動詞の意味と文脈}

【文法ポイント】
助動詞 + 動詞の原形

【注意点】
{その助動詞特有の注意事項}
```

**適用例:**
```markdown
can が正解です。

【理由】
「〜できる」という能力を表す助動詞です。

【文法ポイント】
can + 動詞の原形（can play）

【注意点】
・canの後は必ず原形（plays×）
・疑問文: Can + 主語 + 動詞原形?
・否定文: cannot / can't
```

## 3. verbForm（動詞変化問題）

### 3-1. 規則変化パターン

```markdown
{元の形} → {変化後の形} です。

【変化ルール】
{規則変化のパターン}

【使い方】
{どんな時に使うか}

【類似変化】
{同じパターンの他の動詞}
```

**適用例:**
```markdown
play → played です。

【変化ルール】
規則動詞の過去形: 語尾に-edをつける

【使い方】
過去の動作・状態を表す（昨日〜した、〜だった）

【類似変化】
walk → walked、study → studied（y→iedに注意）
```

### 3-2. 不規則変化パターン

```markdown
{元の形} → {変化後の形} です。

【変化タイプ】
{不規則変化のパターン分類}

【覚え方】
{記憶のヒント}

【例文】
{実際の使用例}
```

**適用例:**
```markdown
go → went です。

【変化タイプ】
A-B-C型の不規則変化（原形・過去形・過去分詞がすべて異なる）
go - went - gone

【覚え方】
頻出動詞なので繰り返し使って体で覚えましょう。
went（行った）、gone（行ってしまった）

【例文】
I went to Tokyo yesterday.（私は昨日東京へ行きました）
```

## 4. paraphrase（言い換え問題）

### 4-1. 時制変換パターン

```markdown
{元の時制} を {変換後の時制} に変換します。

【変換ポイント】
・動詞: {変化の説明}
・時を表す語: {変化の説明}

【意味の変化】
{時制変換による意味の違い}
```

**適用例:**
```markdown
現在形を過去形に変換します。

【変換ポイント】
・動詞: play → played
・時を表す語: every day → yesterday

【意味の変化】
現在形: 習慣・反復（毎日している）
過去形: 過去の一時点（昨日した）
```

### 4-2. 構文変換パターン

```markdown
{元の構文} を {変換後の構文} に言い換えます。

【変換方法】
{具体的な変換手順}

【注意点】
{意味が変わらないための注意}

【比較】
元の文: {元の文}
変換後: {変換後の文}
```

**適用例:**
```markdown
肯定文を否定文に言い換えます。

【変換方法】
1. 動詞の前にdon't/doesn'tを置く
2. 動詞を原形に戻す

【注意点】
三人称単数の場合はdoesn'tを使用

【比較】
元の文: He plays tennis.
変換後: He doesn't play tennis.
```

## 5. conversation（会話問題）

### 5-1. 応答選択パターン

```markdown
{正解選択肢} が正解です。

【文脈理解】
{会話の流れと状況}

【文法ポイント】
{使用されている文法}

【他の選択肢】
{なぜ他の選択肢が不適切か}
```

**適用例:**
```markdown
"Yes, I do." が正解です。

【文脈理解】
"Do you like music?"（音楽は好きですか？）という一般動詞の疑問文への応答です。

【文法ポイント】
一般動詞の疑問文への応答: Yes, I do. / No, I don't.

【他の選択肢】
× "Yes, I am.": be動詞の疑問文への応答
× "Yes, I can.": 助動詞canの疑問文への応答
```

### 5-2. 対話補完パターン

```markdown
{正解表現} が適切です。

【場面設定】
{会話が行われている状況}

【表現のポイント】
{その表現を使う理由}

【類似表現】
{同じ場面で使える他の表現}
```

**適用例:**
```markdown
"How are you?" が適切です。

【場面設定】
挨拶の場面で、相手の様子を尋ねる定型表現です。

【表現のポイント】
親しい間柄での日常的な挨拶として最も一般的

【類似表現】
"How's it going?"（調子はどう？）- よりカジュアル
"How do you do?"（初対面の挨拶）- フォーマル
```
