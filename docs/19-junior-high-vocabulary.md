# 19. 中学語彙データセット仕様書

## 📚 概要

中学英語の基礎となる重要単語1000語のデータセットです。教科書頻出語彙を中心に、品詞・難易度・カテゴリーで体系的に分類されています。

---

## 🎯 データ仕様

### 1. ファイル形式

- **フォーマット**: CSV（7列）
- **エンコーディング**: UTF-8
- **総語数**: 約1000語
- **ファイル**: `public/data/junior-high-*.csv`

### 2. 列定義

```
語句,読み,意味,語源等解説,関連語,関連分野,難易度
```

### 3. データ例

```csv
study,スタディ,勉強する,ラテン語studiumから,student(学生),教育,初級
important,インポータント,重要な,import(輸入)+ -ant,importance,一般,中級
```

### 4. カテゴリー

- 動詞
- 名詞
- 形容詞
- 副詞
- 前置詞

### 5. 難易度

- 初級: 中1レベル
- 中級: 中2-3レベル
- 上級: 高校入試レベル
