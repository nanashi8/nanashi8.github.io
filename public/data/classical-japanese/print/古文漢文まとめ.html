漢文重要語句

【返り点】

一二点 【いちにてん】 ★
漢文の返り点の一つ。「一」を先に「二」を後に読む
返り点は漢文を日本語の語順で読むための記号。一二点は近い距離の返り読みに使用する。例：「読書一」→「書を読む」。レ点よりも離れた文字を返す際に用いる。
上下点 【じょうげてん】 ★
漢文の返り点の一つ。「上」を先に「下」を後に読む
一二点よりも広い範囲の返り読みに使用する返り点。「上」→「下」の順に読む。さらに広範囲の場合は甲乙点（甲→乙）を使用する。三つ以上の返り読みがある場合に階層的に使い分ける。
甲乙点 【こうおつてん】 ★★
漢文の返り点の一つ。「甲」を先に「乙」を後に読む
上下点よりもさらに広い範囲の返り読みに使用する返り点。「甲」→「乙」→「丙」→「丁」の順。最も外側の大きな括りとして使用される。複雑な文構造で複数の返り読みを整理するために必要。
レ点 【れてん】 ★
すぐ下の字を先に読んで返る記号
最も基本的な返り点。すぐ下の一字だけを返して読む。例：「見レ月」→「月を見る」。レは「返」の草書体に由来。隣接した二字の語順を入れ替える最小単位の返り点。

【再読文字】

未 【いまダ〜ず】 ★
まだ〜しない（未然の否定）
再読文字の一つ。一度目は「いまだ」、二度目は「ず」と読む。例：「未レ知」→「いまだ知らず」。まだその行為や状態に至っていないことを表す。否定の副詞「未だ」と否定の助動詞「ず」の組み合わせ。
将 【まさニ〜んとス】 ★
今にも〜しようとする
再読文字の一つ。一度目は「まさに」、二度目は「〜んとす」と読む。例：「将レ帰」→「まさに帰らんとす」。これから行おうとする意志や、そうなりそうな状態を表す。推量の意味を含む。
当 【まさニ〜べシ】 ★★
当然〜すべきである
再読文字の一つ。一度目は「まさに」、二度目は「べし」と読む。例：「当レ行」→「まさに行くべし」。義務や当然性を表す。道理として行うべきことを示す。
須 【すべから〜べシ】 ★★
ぜひとも〜すべきである
再読文字の一つ。一度目は「すべからく」、二度目は「べし」と読む。例：「須レ学」→「すべからく学ぶべし」。強い義務や必要性を表す。「当」よりも強い意味合い。
宜 【よろシ〜べシ】 ★★
〜するのがよい
再読文字の一つ。一度目は「よろしく」、二度目は「べし」と読む。例：「宜レ早」→「よろしく早くすべし」。適切さや妥当性を表す。「〜するのが適当だ」という意味。
猶 【なホ〜ごとシ】 ★★★
なお〜のようだ
再読文字の一つ。一度目は「なお」、二度目は「ごとし」と読む。例：「猶レ昔」→「なお昔のごとし」。依然として前の状態が続いていることを表す。比喩や例えを伴うことが多い。
盍 【なんゾ〜ざル】 ★★★
どうして〜しないのか
反語を表す再読文字。一度目は「なんぞ」、二度目は「ざる」と読む。例：「盍レ帰」→「なんぞ帰らざる」。「〜しないのはおかしい」という意味で、実質的には勧誘を表す。

【否定】

不 【〜ず】 ★
〜ない（打消）
否定を表す基本的な文字。「ず」と読む。例：「不レ知」→「知らず」。最も一般的な否定表現。英語のnotに相当。動詞や形容詞を否定する。
非 【〜にあらず】 ★
〜ではない
名詞や代名詞を否定する文字。「あらず」と読む。例：「非レ人」→「人にあらず」。「不」は動詞否定、「非」は名詞否定と使い分ける。英語のis notに相当。
無 【〜なシ】 ★
〜がない、〜しない
存在の否定や動作の否定を表す。「なし」と読む。例：「無レ友」→「友なし」。持っていない、存在しないという意味。「不」より存在自体の否定が強い。

【禁止】

莫 【〜なかレ】 ★★
〜するな（禁止）
禁止を表す文字。「なかれ」と読む。例：「莫レ笑」→「笑うなかれ」。相手に対して行為を禁止する命令文。「勿」も同様の用法。
勿 【〜なかレ】 ★★
〜するな（禁止）
禁止を表す文字。「なかれ」と読む。例：「勿レ言」→「言うなかれ」。「莫」とほぼ同じ意味。やや柔らかい禁止表現として使われることもある。

【比喩】

如 【〜ごとシ】 ★
〜のようだ
比喩・比況を表す文字。「ごとし」と読む。例：「如レ雲」→「雲のごとし」。「AがBのようだ」という直喩表現。類似性を示す。
若 【〜ごとシ】 ★★
〜のようだ
比喩を表す文字。「ごとし」と読む。例：「若レ夢」→「夢のごとし」。「如」とほぼ同義。やや文語的で格調高い表現。

【前置詞】

於 【〜ニ】 ★
〜に、〜において
場所や時を表す前置詞。「に」「において」と読む。例：「生於レ貧」→「貧しきに生まる」。動作の場所や時点を示す。英語のin, at, onに相当。

【助詞】

乎 【〜か】 ★
〜だろうか（疑問・反語）
文末の疑問を表す助詞。「か」と読む。例：「知乎」→「知るか」。疑問文や反語文を作る。感嘆を表すこともある。
哉 【〜かな】 ★
〜だなあ（詠嘆）
感動や詠嘆を表す助詞。「かな」と読む。例：「美哉」→「美しきかな」。深い感動や感慨を表現する。漢詩の結句によく使われる。
矣 【〜のみ】 ★★
〜だけだ、〜である
断定や限定を表す助詞。「のみ」「り」と読む。例：「足矣」→「足れり」。強い断定や完了を表す。文末に置かれることが多い。
也 【〜なり】 ★
〜である
断定を表す助詞。「なり」と読む。例：「学者也」→「学者なり」。主語と述語を結ぶ判断の助詞。英語のisに相当。
耶 【〜や】 ★★
〜か（疑問）
疑問を表す助詞。「や」と読む。例：「然耶」→「然るや」。「乎」と同様に疑問文を作る。やや口語的な印象。
焉 【〜たり】 ★★★
〜である（断定）
文末で断定を表す場合もある。「たり」と読む。多機能な文字で、文脈により意味が変わる。助詞としても代名詞としても使われる。

【代名詞】

焉 【これ、いづくんゾ】 ★★★
これ、どうして
指示代名詞または疑問詞。文脈により読み分ける。例：「何為焉」→「何をかこれ為さん」。多義的で文脈判断が重要。
其 【その】 ★
その
指示代名詞。「その」と読む。例：「其人」→「その人」。三人称の所有を表すことが多い。英語のhis, her, itsに相当。

【代名詞・助詞】

之 【これ、の】 ★
これ、〜の
代名詞「これ」または助詞「の」。例：「読二之一」→「これを読む」、「天之道」→「天の道」。漢文で最も頻出する文字の一つ。

【疑問詞】

安 【いづくんゾ】 ★★
どうして、どこに
疑問詞。「いずくんぞ」「いずくにか」と読む。例：「安得」→「いずくんぞ得ん」。場所や方法を問う。反語を表すことも多い。
何 【なニ、なんゾ】 ★
何、なぜ
疑問詞。「なに」「なんぞ」と読む。例：「何為」→「何をか為さん」。事物や理由を問う。英語のwhatやwhyに相当。
胡 【なんゾ】 ★★★
どうして
疑問詞。「なんぞ」と読む。例：「胡不レ帰」→「なんぞ帰らざる」。理由を問う。反語を表すことが多い。

【故事成語】

矛盾 【むじゅん】 ★
つじつまが合わないこと
「矛」は攻撃の武器、「盾」は防御の武器。「どんな盾も突き通す矛」と「どんな矛も防ぐ盾」を同時に売ろうとした商人の話に由来。韓非子の説話。論理的な矛盾を指す。
推敲 【すいこう】 ★
文章を練り直すこと
唐の詩人賈島が「僧推月下門」か「僧敲月下門」か迷い、韓愈に相談した故事に由来。「推す」か「敲く（たたく）」かで悩んだことから、詩文を何度も練り直すことを意味する。
蛇足 【だそく】 ★
余計なもの、無用の長物
「蛇を描くのに足を加える」という故事に由来。酒を早く飲める者に与えるという競争で、先に描き終えた者が「足も描こう」と余計なことをした話（戦国策）。必要のない付け足しを指す。
漁夫之利 【ぎょふのり】 ★★
第三者が争いの利益を得ること
シギと二枚貝が争っている間に漁師が両方とも捕まえた故事（戦国策）。二者が争っている隙に第三者が利益を得ることを意味する。類義語：両者争えば両者失う。

【漢詩】

五言絶句 【ごごんぜっく】 ★
五字四句の漢詩形式
一句が五字、全体で四句（二十字）の漢詩。起句・承句・転句・結句の構成。押韻は通常第二句と第四句。最も基本的な近体詩の形式。
七言絶句 【しちごんぜっく】 ★
七字四句の漢詩形式
一句が七字、全体で四句（二十八字）の漢詩。五言絶句より表現が豊かになる。杜甫「春望」など名作多数。押韻規則は五言絶句と同じ。
五言律詩 【ごごんりっし】 ★★
五字八句の漢詩形式
一句が五字、全体で八句（四十字）の漢詩。絶句より長く、複雑な内容を表現できる。中四句で対句を用いる規則がある。格調高い作品が多い。
七言律詩 【しちごんりっし】 ★★
七字八句の漢詩形式
一句が七字、全体で八句（五十六字）の漢詩。律詩の中で最も長大な形式。対句や押韻の規則が厳格。杜甫が名手として知られる。
起句 【きく】 ★
漢詩の第一句
詩の導入部分。情景や主題を提示する。読者を詩の世界に誘い込む役割。場面設定や時間・場所の提示が多い。
承句 【しょうく】 ★★
漢詩の第二句
起句を受けて展開する句。主題をさらに深める。起句の内容を承けて具体化したり、説明を加えたりする。
転句 【てんく】 ★★
漢詩の第三句
詩の視点や内容を転換する句。新しい視点や対比を提示。詩に変化と深みを与える最も重要な句。意外性や発見がある。
結句 【けっく】 ★
漢詩の第四句（最終句）
詩を締めくくる句。主題を総括し、余韻を残す。感慨や教訓を述べることが多い。読者の心に残る印象的な表現が求められる。
対句 【ついく】 ★★
対になる二つの句
意味や文法構造が対応する二句の組。例：「天高く地低し」。律詩では中四句（第三・四句と第五・六句）で対句を用いる規則。美的効果を高める修辞技法。
押韻 【おういん】 ★★
句末の音を揃えること
詩の各句の最後の字の音（韻）を揃える技法。リズムと音楽性を生む。近体詩では偶数句（第二・四句など）の末尾で韻を踏む規則がある。
平仄 【ひょうそく】 ★★★
漢字の音調
漢字の声調を「平声」と「仄声」に分類したもの。近体詩では平仄の配置に厳格な規則がある。音楽的なリズムを作る基礎。日本人には習得が難しい。
近体詩 【きんたいし】 ★★
唐代に完成した定型詩
絶句と律詩の総称。平仄・押韻・対句などの厳格な規則がある。唐代に完成された洗練された詩形。格調高い作品が多い。
古体詩 【こたいし】 ★★★
形式が自由な漢詩
近体詩以前の自由な形式の詩。句数や字数に制限が少ない。平仄の規則も緩やか。率直で力強い表現が特徴。
絶句 【ぜっく】 ★
四句からなる短詩
近体詩の一形式。五言絶句と七言絶句がある。短い中に深い情趣を込める。最も親しまれている漢詩の形式。
律詩 【りっし】 ★★
八句からなる定型詩
近体詩の一形式。五言律詩と七言律詩がある。対句や押韻の規則が厳格。格調高く複雑な内容を表現できる。

古文文法・助動詞

たまふ 【たまう】
お与えになる・〜なさる
「たまふ」は尊敬語で「お与えになる」「〜なさる」の意。補助動詞として「〜なさる」の意でも使う。「賜ふ」「給ふ」と書く。最頻出の尊敬語。
尊敬語 / お与えになる / なさる / 最頻出 / 時代:平安時代 / 種別:古文敬語
きこゆ 【きこゆ】
申し上げる・お聞こえになる
「きこゆ」は謙譲語で「申し上げる」の意。尊敬語で「お聞こえになる」の意もある。「聞こゆ」と書く。謙譲と尊敬の両方の用法がある重要語。
謙譲語 / 尊敬語 / 申し上げる / 聞こえる / 時代:平安時代 / 種別:古文敬語
はべり 【はべり】
あります・ございます
「はべり」は丁寧語で「あります」「ございます」「お仕えする」の意。「侍り」と書く。「あり」の丁寧語。会話文に頻出。
丁寧語 / あります / ございます / 侍り / 時代:平安時代 / 種別:古文敬語
さぶらふ 【さぶらう】
お仕えする・おります
「さぶらふ」は謙譲語で「お仕えする」「おります」の意。「候ふ」と書く。貴人の近くに控える意味。「はべり」より古風。
謙譲語 / お仕えする / おります / 候ふ / 時代:平安時代 / 種別:古文敬語
おはす 【おわす】
いらっしゃる・おいでになる
「おはす」は尊敬語で「いらっしゃる」「おいでになる」「おられる」の意。「御座す」と書く。「あり」「をり」「行く」「来」の尊敬語。最重要尊敬語。
尊敬語 / いらっしゃる / おいでになる / 御座す / 時代:平安時代 / 種別:古文敬語
まかる 【まかる】
退出する・参る
「まかる」は謙譲語で「退出する」「参る」の意。「罷る」と書く。貴人の前から退く意味。「行く」の謙譲語としても使う。
謙譲語 / 退出する / 参る / 罷る / 時代:平安時代 / 種別:古文敬語
まかづ 【まかず】
退出させる・お帰しになる
「まかづ」は「退出させる」「お帰しになる」の意。「まかる」の使役形。「罷づ」と書く。貴人が人を退出させる意味。
退出させる / 帰す / 使役 / 罷づ / 時代:平安時代 / 種別:古文敬語
まゐる 【まいる】
参上する・差し上げる
「まゐる」は謙譲語で「参上する」「差し上げる」の意。「参る」と書く。「行く」「来」「食ふ」「飲む」などの謙譲語。最頻出謙譲語。
謙譲語 / 参上する / 差し上げる / 参る / 時代:平安時代 / 種別:古文敬語
たてまつる 【たてまつる】
差し上げる・申し上げる
「たてまつる」は謙譲語で「差し上げる」「申し上げる」の意。「奉る」と書く。「〜してさしあげる」の補助動詞としても頻出。最重要謙譲語。
謙譲語 / 差し上げる / 申し上げる / 奉る / 時代:平安時代 / 種別:古文敬語
けり 【けり】
〜た・〜たなあ
「けり」は過去・詠嘆の助動詞。「〜た」「〜たなあ」の意。ラ変型活用。連用形接続。伝聞過去や詠嘆を表す。最重要助動詞の一つ。
過去 / 詠嘆 / 助動詞 / 伝聞過去 / 時代:平安時代 / 種別:助動詞
き 【き】
〜た
「き」は過去の助動詞。「〜た」の意。特殊型活用。連用形接続。直接過去(自分が体験した過去)を表す。「けり」(伝聞過去)と対比される重要助動詞。
過去 / 直接過去 / 体験 / けりと対 / 時代:平安時代 / 種別:助動詞
つ 【つ】
〜てしまった・〜た
「つ」は完了の助動詞。「〜てしまった」「〜た」の意。下二段型活用。連用形接続。強い完了・確述を表す。「ぬ」と対で重要助動詞。
完了 / 確述 / てしまった / 強意 / 時代:平安時代 / 種別:助動詞
ぬ 【ぬ】
〜てしまった・〜た
「ぬ」は完了の助動詞。「〜てしまった」「〜た」の意。ナ変型活用。連用形接続。完了・確述を表す。「つ」と対。打消の「ず」の連体形とは別。
完了 / 確述 / てしまった / つと対 / 時代:平安時代 / 種別:助動詞
たり 【たり】
〜ている・〜だ
「たり」は完了・存続の助動詞。「〜ている」「〜だ」の意。ラ変型活用。連用形(音便)接続。状態の継続を表す。「咲きたり」などの形。
完了 / 存続 / ている / 音便接続 / 時代:平安時代 / 種別:助動詞
り 【り】
〜ている・〜た
「り」は完了・存続の助動詞。「〜ている」「〜た」の意。ラ変型活用。サ変の未然形・四段の已然形に接続。特殊な接続をする助動詞。
完了 / 存続 / ている / 特殊接続 / 時代:平安時代 / 種別:助動詞
ず 【ず】
〜ない
「ず」は打消の助動詞。「〜ない」の意。特殊型活用。未然形接続。連体形は「ぬ」、已然形は「ね」。打消表現として最重要。「〜ず」「〜ぬ」「〜ね」の使い分けに注意。
打消 / ない / ぬね / 未然形接続 / 時代:平安時代 / 種別:助動詞
む 【む】

「なり」「たり」は断定を表す助動詞。「花なり」「学者たり」（花だ・学者だ）。「なり」は体言・連体形接続、「たり」は体言接続。現代語の「だ」に相当。
