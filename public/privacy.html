プライバシーポリシー

最終更新日: 2025年11月22日

1. はじめに

nanashi8-quiz-app（以下「本サービス」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本サービスがどのような情報を収集し、どのように使用するかを説明します。

2. 収集する情報

2.1 自動収集される情報

  • 学習データ: クイズの進捗状況、正答率、学習履歴はブラウザのローカルストレージに保存されます
    • LocalStorage: 設定情報、一部の進捗データ
    • IndexedDB: クイズ履歴、学習進捗、カスタム問題集など大容量データ
    • データは自動的にLocalStorageからIndexedDBに移行される場合があります
  • エラー情報: アプリケーションのエラー発生時、以下の技術的情報がSentry（エラー監視サービス）に送信されます
    • エラーメッセージとスタックトレース
    • ブラウザの種類とバージョン
    • OSの種類とバージョン
    • 画面解像度
    • エラー発生時のページURL
    • パフォーマンス計測データ（10%サンプリング）
    • セッションリプレイ（エラー発生時のみ記録）
  • アクセスログ: GitHub Pagesによるホスティングにより、以下の情報がGitHubのサーバーに記録される場合があります
    • IPアドレス
    • アクセス日時
    • ユーザーエージェント（ブラウザ情報）

2.2 収集しない情報

本サービスは以下の情報を収集しません:

  • 氏名、メールアドレス、電話番号などの個人を特定できる情報
  • クレジットカード情報などの決済情報
  • 位置情報（GPS等）
  • Cookie（現時点では使用していません）
  • 学習データの内容（LocalStorage/IndexedDBの内容は外部送信されません）

3. 情報の使用目的

  • 学習データ: ユーザーの学習進捗を保存し、次回アクセス時に継続できるようにするため
  • エラー情報: アプリケーションの品質向上、バグ修正、パフォーマンス改善のため
  • アクセスログ: サービスの安定運用、セキュリティ対策のため（GitHub Pagesによる収集）

4. 情報の保存場所と保持期間

4.1 保存場所

  • 学習データ: ユーザーのブラウザ内（LocalStorageおよびIndexedDB）にのみ保存され、外部サーバーには送信されません
  • エラー情報: Sentry.io（アメリカ合衆国）のサーバーに送信されます
  • アクセスログ: GitHub（アメリカ合衆国）のサーバーに記録されます

4.2 保持期間

  • 学習データ: ユーザーが削除するまで無期限に保存されます
  • エラーログ: Sentryで90日間保存後、自動削除されます
  • アクセスログ: GitHub Pagesのポリシーに準拠します

5. 第三者サービスの利用

本サービスは、以下の第三者サービスを利用しています:

5.1 Sentry（エラー監視）

  • 提供者: Functional Software, Inc.
  • 所在地: アメリカ合衆国
  • 目的: エラー監視、パフォーマンス計測
  • プライバシーポリシー: https://sentry.io/privacy/

5.2 GitHub Pages（ホスティング）

  • 提供者: GitHub, Inc.
  • 所在地: アメリカ合衆国
  • 目的: Webサイトのホスティング
  • プライバシーポリシー: GitHub Privacy Statement
  • GitHub Pagesについて: データ収集に関する情報

5.3 Web Speech API（音声合成）

  • 提供者: ブラウザベンダー（Chrome、Safari、Firefox等）
  • 目的: 英単語・英文の発音機能
  • データ処理: 基本的にブラウザ内で処理されますが、一部のブラウザ（Chrome等）では音声合成処理のためにGoogleサーバーと通信する場合があります
  • Chrome音声機能: https://support.google.com/chrome/answer/95884

6. 国際データ転送

本サービスは、以下のデータをアメリカ合衆国のサーバーに転送します:

  • エラー情報（Sentryサーバー）
  • アクセスログ（GitHub Pagesサーバー）

これらのサービス提供者は、適切なデータ保護対策を実施しています。EUユーザーの場合、標準契約条項（SCC）等の適切な保護措置が適用されます。

7. 情報の共有

本サービスは、以下の場合を除き、収集した情報を第三者と共有しません:

  • 法令に基づく開示要求がある場合
  • サービスの運営に必要な範囲で、上記第三者サービス（Sentry、GitHub Pages）と共有する場合
  • ユーザーの同意がある場合

8. ユーザーの権利

  • データの閲覧: アプリ内の設定画面から、保存されている学習データを確認できます
  • データのエクスポート: アプリ内の設定画面から、学習データをJSON形式でダウンロードできます
  • データの削除:
    • アプリ内: 設定画面から学習データを削除できます
    • ブラウザ: ブラウザの設定から本サイトのデータ（LocalStorage/IndexedDB）を完全に削除できます
    • Sentryのデータ: GitHubリポジトリのIssuesから削除リクエストを送信してください
  • データポータビリティ: エクスポートしたJSONデータは他のアプリケーションでも利用可能です
  • 苦情申し立て: プライバシーに関する懸念がある場合、お住まいの地域のデータ保護機関に申し立てることができます

7. セキュリティ

本サービスは以下のセキュリティ対策を実施しています:

  • HTTPS通信による暗号化（GitHub Pages）
  • エラー監視サービス（Sentry）はSOC 2 Type II認証取得済み
  • 個人情報を収集しないことによるリスク最小化

8. 子供のプライバシー

本サービスは中学受験を目指す児童を含む全年齢のユーザーを対象としていますが、個人情報を収集しないため、特別な保護者同意は不要です。

9. プライバシーポリシーの変更

本プライバシーポリシーは、サービスの改善に伴い変更される場合があります。重要な変更がある場合は、本ページ上部の「最終更新日」を更新し、お知らせします。

10. お問い合わせ

プライバシーポリシーに関するご質問は、GitHubリポジトリ（nanashi8/nanashi8.github.io）のIssuesにてお問い合わせください。
