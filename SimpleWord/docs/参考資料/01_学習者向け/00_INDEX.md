## ❓ よくある質問

### Q1: どのくらいの学習時間が必要？
**A**: 合計45分程度です
- クイックスタートガイド: 15分
- アーキテクチャ入門: 30分

### Q2: プログラミング初心者でも大丈夫？
**A**: Swift と SwiftUI の基礎知識があれば理解できます
- Swift の基本構文（変数、関数、クラス）
- SwiftUI の基本（View, @State, @ObservedObject）
