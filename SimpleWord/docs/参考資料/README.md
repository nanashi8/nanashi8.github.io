## 🚀 はじめに

### 新規参加者向け
1. [クイックリファレンス](./AIとのやり取り_クイックリファレンス.md) を読む（30秒）

1. [INDEX](./AIとのやり取り_INDEX.md) で自分に合ったドキュメントを選ぶ

3. [ケーススタディ集](./AIとのやり取り_ケーススタディ集.md) で効果を確認

1. [機能復元ガイド_v2.md](./機能復元ガイド_v2.md) を開く
2. 優先度別復元計画を確認
3. 該当する仕様書を参照（`docs/仕様書/`）
4. 段階的に復元

## 💡 おすすめの使い方

### バグ報告時
1. [クイックリファレンス](./AIとのやり取り_クイックリファレンス.md) を開く
2. 「バグ報告」テンプレートをコピー
3. [ ] 内を埋める
4. AI に送信

**所要時間**: 1分

### 機能追加時
1. [テンプレート集](./効率的なやり取りテンプレート集.md) を開く
2. 「新機能追加依頼」セクションを参照
3. 仕様を明確化
4. AI に依頼

**所要時間**: 5分

[ケーススタディ集](./AIとのやり取り_ケーススタディ集.md) で、平均 64% の時間削減効果を確認できます。
