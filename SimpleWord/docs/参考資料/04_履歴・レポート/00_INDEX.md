# SimpleWord

**最終更新**: 2025年11月3日  
**バージョン**: 1.0

SwiftUI + Combineで構築された、CSV駆動型の学習アプリケーションです。

---

## 🎯 概要

SimpleWordは、CSVファイルから学習コンテンツを読み込み、適応型学習アルゴリズムに基づいて最適な出題を行う学習支援アプリです。

### 主な機能
- 📝 **CSV駆動型**: CSVファイルから単語・英会話・古典などの学習コンテンツを読み込み
- 🧠 **適応型学習**: 記憶定着段階に応じた最適な出題タイミング
- 📊 **学習進捗管理**: 正答率・学習履歴・記憶段階を可視化
- ⚙️ **柔軟な設定**: 出題範囲・難易度・学習モードのカスタマイズ
- 🎨 **直感的UI**: SwiftUIによるシンプルで使いやすいインターフェース
