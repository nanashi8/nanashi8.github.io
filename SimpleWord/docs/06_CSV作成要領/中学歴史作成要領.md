# 中学歴史問題CSV作成・拡充Instruction

この文書は、プロジェクトで使用する `中学歴史.csv` を一貫して、かつ手作業で作成・拡充しやすくするための具体的な作成要領です。

## 目的
- 中学校で学ぶ日本史・世界史の重要な史実を、年代・出来事・原因を理解しやすい形で整備する
- 史実の背景・原因・結果の因果関係を明確にし、記憶定着率を向上させる
- 語呂合わせや視覚的イメージを活用し、年号と出来事を効率的に覚えられるようにする
- **目標**: 中学校歴史の重要史実300個以上を網羅的に収録する

## 収録対象の時代区分と目標数

### 日本史（240個程度）
1. **原始・古代**（40個）: 縄文〜奈良時代
   - 旧石器・縄文・弥生時代、古墳時代、飛鳥時代、奈良時代
2. **平安時代**（20個）: 平安京遷都〜院政
3. **中世**（40個）: 鎌倉〜室町・戦国時代
   - 鎌倉幕府、元寇、南北朝、室町幕府、応仁の乱、戦国時代
4. **近世**（40個）: 安土桃山〜江戸時代
   - 信長・秀吉、江戸幕府、鎖国、元禄・化政文化、幕末
5. **近代**（60個）: 明治〜昭和戦前
   - 明治維新、自由民権運動、大日本帝国憲法、日清・日露戦争、第一次世界大戦、大正デモクラシー、昭和恐慌、満州事変、日中戦争、太平洋戦争
6. **現代**（40個）: 昭和戦後〜平成・令和
   - 占領期、冷戦、高度経済成長、バブル経済、平成・令和の出来事

### 世界史（60個程度）
1. **古代文明**（10個）: 四大文明、ギリシャ・ローマ
2. **中世ヨーロッパ**（10個）: キリスト教、十字軍、ルネサンス
3. **大航海時代〜近世**（15個）: 大航海時代、宗教改革、絶対王政
4. **市民革命〜産業革命**（15個）: イギリス・アメリカ・フランス革命、産業革命
5. **帝国主義〜世界大戦**（10個）: 帝国主義、第一次・第二次世界大戦

## 収録の優先順位

### 最優先（必須）
- 教科書に太字で記載される重要事項
- 年号暗記が必要な史実（語呂合わせがあるもの）
- 入試頻出の史実

### 高優先
- 教科書の本文で詳しく説明される事項
- 歴史の流れを理解する上で重要な転換点
- 文化史の代表的作品・人物

### 中優先
- 教科書の側注や資料で触れられる事項
- 地域史・テーマ史の重要事項
- 人物の業績

### 低優先（余裕があれば）
- 詳細な年号や細かい出来事
- 発展的な内容

## 前提
- CSVはターミナルを使わず直接編集しなさい。
- 文字コード: UTF-8（BOM なし）
- 区切り文字: カンマ ( , )
- 改行: LF 推奨
- ファイル先頭に必ずヘッダ行を残す（下記ヘッダ順を厳守）

## 必須ヘッダ（順序固定）
1. 年号
2. 登場人物
3. 史実名
4. 解説
5. 関連史実
6. 関連分野
7. 難易度

## 各列の定義と入力ルール

### 1. 年号
- **西暦を基本とし、和暦を併記**
- フォーマット: `西暦年 (和暦)` または `西暦年`
- 和暦は主要なもののみ記載（明治以降、江戸時代の重要な元号等）
- 世紀単位の場合は「〜世紀頃」「〜世紀」と記載

**例**:
```
1192
1603 (慶長8年)
1868 (明治元年)
1945 (昭和20年)
4世紀頃
19世紀初頭
```

### 2. 登場人物
- その史実に関わる主要な人物を記載
- 複数の場合は「・」で区切る（2〜3人程度）
- 天皇、将軍、武将、文化人など

**例**:
```
聖徳太子・蘇我馬子・推古天皇
中大兄皇子・中臣鎌足
源頼朝・源義経
徳川家康
坂本龍馬・西郷隆盛・木戸孝允
```

### 3. 史実名
- **何が起こったかを簡潔に記載**（30文字以内を目安）
- 主語と述語を明確にする
- 固有名詞は正確に記載
- **重要**: この列が問題として出題されます

**良い例**:
```
大和政権の成立
聖徳太子の摂政
十七条憲法の制定
大化の改新
鎌倉幕府の成立
```

**避けるべき例**:
```
いい国作ろう  // 史実ではなく語呂合わせ
武士の時代が始まる  // 抽象的すぎる
```

### 4. 解説
- **目的**: その史実が起こった原因・背景・経緯を説明し、因果関係を理解することで記憶を助ける
- **文字数**: 40〜200 文字程度（簡潔かつ具体的に）
- **必須項目**: この欄を充実させることが記憶定着の鍵
- **重要**: この列が選択肢として表示されます

**記述内容（優先順）**:

1. **直接的な原因・きっかけ**（最優先）
   - 例: "平氏が権力を握り武士が台頭。源氏と平氏の対立が激化し、源平合戦で源氏が勝利したため"

3. **国際的な影響・外圧**
   - 例: "ヨーロッパでナポレオン戦争。日本も国際情勢の変化に巻き込まれる"
   - 例: "アメリカが太平洋進出を目指す。日本を中継地点として開国を迫った"
   - 例: "第一次世界大戦で欧米が疲弊。日本が中国進出を加速させる好機となった"

4. **人物の動機・思想**
   - 例: "織田信長が本能寺の変で討たれる。秀吉は主君の仇を討ち、天下統一を目指した"
   - 例: "坂本龍馬が薩長同盟を仲介。幕府を倒し新しい国づくりを目指した"
   - 例: "西郷隆盛が征韓論を主張。しかし明治政府は内政優先を選択し対立"

5. **前後の史実との因果関係**
   - 例: "応仁の乱で京都が荒廃→戦国時代の始まり→織田信長の登場"
   - 例: "日露戦争の勝利→韓国併合→満州進出→日中戦争"
   - 例: "黒船来航→開国→尊王攘夷運動→大政奉還→明治維新"

6. **対立構造・勢力図**
   - 例: "朝廷（天皇）vs 幕府（将軍）の権力争い"
   - 例: "保守派（幕府支持）vs 革新派（倒幕派）の対立"
   - 例: "軍部（拡大路線）vs 政党（協調路線）の対立"

**記述の原則**:
- ✓ 「なぜその史実が起こったのか」を明確にする
- ✓ 直接的な原因を最初に述べる
- ✓ 背景→原因→結果の流れを意識する
- ✓ 複数の要因がある場合は優先順位をつける
- ✓ 人物名や具体的な出来事を挙げる
- ✓ 「～だったため」「～の結果」など因果関係を示す語を使う

**避けるべき説明**:
- ✗ 史実の単なる繰り返し（例: "関ヶ原で戦った"だけ）
- ✗ 抽象的すぎる説明（例: "時代の流れで"）
- ✗ 原因が不明確（例: "色々な理由で"）
- ✗ 史実の説明だけで経緯がない

**良い例**:
```
平清盛が武士として初めて太政大臣となり平氏政権を樹立。しかし平氏の専横に不満を持った源氏が挙兵。源平合戦で源氏が勝利し、武士が政治の実権を握る契機となった。

ペリーが浦賀に来航し開国を要求。幕府は250年続いた鎖国政策を転換せざるを得なくなった。開国派と攘夷派が対立し、政治的混乱が深まった結果、倒幕運動へとつながる。

関東大震災で甚大な被害。政府の対応の遅れと混乱の中で、朝鮮人虐殺などの社会不安が発生。震災恐慌により経済も悪化し、軍部台頭の一因となった。
```

### 5. 関連史実
- フォーマット: `史実:説明` のペアをセミコロン (;) で区切る
- 時系列的に前後する史実や、因果関係のある史実を3〜5個程度

**例**:
```
古墳文化;律令以前の統合過程
十七条憲法(604);遣隋使
推古朝の改革
皇位と律令国家の成立過程
律令制;班田収授法
```

### 6. 関連分野
以下の20カテゴリから**必ず1つ**を選択（歴史分野では主に「文化」「その他」を使用）:

1. 日常
2. 学校
3. 家族
4. 食べ物
5. 交通
6. 旅行
7. 健康
8. 動物
9. 自然
10. 地理
11. 科学
12. 文法
13. 文化  ← 歴史・文化史
14. 時間
15. テクノロジー
16. 買い物
17. 職業
18. スポーツ
19. 感情
20. その他  ← 政治史・戦争史

**推奨カテゴリ**:
- 政治・戦争: `その他`
- 文化・芸術: `文化`
- 地理的な出来事: `地理`
- 科学技術: `科学` または `テクノロジー`

### 7. 難易度
- **1**: 基礎レベル（教科書の太字、重要年号）
- **2**: 標準レベル（教科書本文の詳細、文化史）
- **3**: 発展レベル（高校受験、複雑な因果関係）

## クイズアプリでの表示

### 出題形式
- **問題**: 「年号」列が表示されます（例: 「794 (延暦13年)」）
- **選択肢**: 「史実名」列から4択で出題されます
- **正解**: その年号に対応する「史実名」列
- **回答後の表示**: 全選択肢に以下が表示されます
  - 年号（問題として出題されたもの）
  - 史実名（選択肢として表示されたもの）
  - 解説
  - 登場人物
  - 関連史実
  - 関連分野
  - 難易度

### 記述の注意点
- **年号**列は問題として表示されるため、明確に記載すること
- **史実名**列は選択肢として表示されるため、他の史実と区別できる明確な名称を使うこと
- 似たような史実名が複数あると、選択肢として区別が難しくなるため注意
- **登場人物**列には主要な人物を2〜3人記載すること
- **解説**列は回答後に詳細情報として表示されます
- **回答後の表示**: 全選択肢に以下が表示されます
  - 年号
  - 語呂合わせ
  - 史実名（問題として出題されたもの）
  - 解説（選択肢として表示されたもの）
  - 関連史実
  - 関連分野
  - 難易度

### 記述の注意点
- **解説**列は選択肢として表示されるため、他の史実と区別できる具体的な内容を記載すること
- 似たような解説が複数あると、選択肢として区別が難しくなるため注意
- **史実名**列は問題として表示されるため、簡潔で分かりやすい表現にすること
- **語呂合わせ**列には年号を覚えるための語呂合わせを優先的に記載すること

## 入力チェックリスト
- [ ] ファイルが UTF-8（BOM なし）で保存されている
- [ ] ヘッダが正しい順序で1行目にある：`年号,登場人物,史実名,解説,関連史実,関連分野,難易度`
- [ ] 各行の列数が7列になっている
- [ ] `年号` が適切な形式（西暦、和暦併記）
- [ ] `登場人物` に主要人物が記載されている
- [ ] `史実名` が簡潔で明確（30文字以内）
- [ ] `解説` に原因・背景が具体的に記載されている（40文字以上）
- [ ] `関連史実` に関連する史実が記載されている（`;` で区切り）
- [ ] `関連分野` が20カテゴリのいずれか1つ
- [ ] `難易度` が 1〜3 のいずれか

1192 (建久3年),源頼朝,鎌倉幕府成立,源平合戦で勝利した源頼朝が、東国を支配する基盤を固める。朝廷から征夷大将軍に任命され、鎌倉に幕府を開く。守護・地頭を全国に配置し、武家による政治体制（幕府）を確立。武士が初めて政権の中心となった。,幕府政治の基本,その他,3
1603 (慶長8年),徳川家康,江戸幕府成立,関ヶ原の戦いで勝利した徳川家康が、朝廷から征夷大将軍に任命される。江戸に幕府を開き、大名を統制する幕藩体制を確立。参勤交代などで大名を監視し、260年続く江戸時代の平和が始まった。,参勤交代・鎖国,その他,3

## 備考
- このCSVは中学校の歴史学習を支援する目的で作成されています
- 史実の解説は教科書や信頼できる資料に基づいて正確に記載してください
- 語呂合わせは一般的に知られているものを優先してください（「鳴くよウグイス平安京」「いい国作ろう鎌倉幕府」など）
- 年号の正確性は特に重要です。複数の資料で確認してください
