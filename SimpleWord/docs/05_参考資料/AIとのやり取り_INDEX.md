## 📖 各ドキュメントの詳細

### 1. クイックリファレンス
**ファイル名**: `AIとのやり取り_クイックリファレンス.md`

**こんな時に**:
- 今すぐ問題を報告したい
- テンプレートを素早く確認したい
- 最小限の情報で始めたい

**特徴**:
- 読了時間: 30秒
- 簡潔な箇条書き
- 状況別のミニテンプレート
- すぐに使える実例

**構成**:
```
1. 3秒でわかるコツ
2. 状況別テンプレート（バグ、変更、削除、追加）
3. 効率化の鉄則
4. よくある質問
```

**おすすめ度**: ★★★★★（初心者・急ぎの人向け）

---

### 2. テンプレート集
**ファイル名**: `効率的なやり取りテンプレート集.md`

**こんな時に**:
- 詳細なテンプレートが欲しい
- 網羅的な情報を知りたい
- プロジェクト標準を作りたい

**特徴**:
- 読了時間: 10分
- 8つの状況に対応
- 具体的な記入例
- ベストプラクティス集

**構成**:
```
1. バグ報告
2. 機能変更依頼
3. 機能削除依頼
4. 新機能追加依頼
5. パフォーマンス問題
6. UI/UX改善
7. リファクタリング依頼
8. コードレビュー依頼
```

**おすすめ度**: ★★★★★（詳細版が欲しい人向け）

---

### 3. ケーススタディ集
**ファイル名**: `AIとのやり取り_ケーススタディ集.md`

**こんな時に**:
- 実例から学びたい
- 効果を数値で確認したい
- 改善前後を比較したい

**特徴**:
- 読了時間: 15分
- 5つの実例
- 改善前後の比較
- 時間削減率の提示（平均64%削減）

**構成**:
```
1. UI 表示順序の不具合（15分→3分、80%削減）
2. パフォーマンス問題（20分→5分、75%削減）
3. 新機能追加（30分→10分、67%削減）
4. 機能削除（10分→5分、50%削減）
5. リファクタリング（60分→30分、50%削減）
```

**おすすめ度**: ★★★★☆（実例重視の人向け）

## 🚀 おすすめの学習順序

### 初心者コース（30分）
1. **クイックリファレンス**を読む（3分）
2. **ケーススタディ集**のケース1を読む（5分）

1. **ケーススタディ集**を全て読む（15分）
2. **テンプレート集**を精読（20分）
3. 過去の問題報告を振り返る（10分）
4. 改善版を作成してみる（15分）

### 上級者コース（90分）
1. 全ドキュメントを精読（30分）
2. 自分のプロジェクト用にカスタマイズ（30分）
3. チーム標準として整備（30分）
