## ❓ よくある質問

### Q1: どのドキュメントから読み始めればいい？

**A**: 役割に応じて以下を推奨します：
- **新規参加者**: プロジェクト概要 → クイックスタートガイド → コーディング規約
- **機能開発者**: 該当の機能仕様書 → アーキテクチャ仕様
- **CSV作成者**: 該当科目のCSV作成要領 → CSV仕様

### Q2: AI（Copilot）の活用方法は？

**A**: [05_参考資料/AIとのやり取り_クイックリファレンス.md](05_参考資料/AIとのやり取り_クイックリファレンス.md) を30秒読むだけで効率が大幅に向上します。

### Q3: 機能を追加・変更したい場合は？

**A**: 以下の流れで進めます：
1. 該当の機能仕様書を確認
2. アーキテクチャ仕様でパターンを確認
3. コーディング規約に従って実装
4. テスト戦略に基づいてテスト作成

### Q4: ドキュメントが見つからない場合は？

**A**: 
1. このINDEXファイル内を検索（Cmd+F）
2. [01_仕様書/機能仕様/99_ファイル索引.md](01_仕様書/機能仕様/99_ファイル索引.md) を確認
3. それでも見つからない場合はドキュメント追加が必要

### Q5: ドキュメントを更新したい場合は？

**A**: 
1. 該当ドキュメントを直接編集
2. 更新日とバージョンを記載
3. 大きな変更の場合は [04_レポート/開発記録/04-01_変更履歴.md](04_レポート/開発記録/04-01_変更履歴.md) にも記録
