## 📖 プロジェクト概要

### プロジェクト名

**SimpleWord** - 適応型学習システム搭載の単語学習アプリケーション

### 目的

SimpleWordは、脳科学と記憶理論に基づく**適応型学習アルゴリズム**を搭載し、効率的な単語学習を実現するiOSアプリケーションです。ユーザーの習熟度に応じて最適な出題順序を自動決定し、記憶定着を促進します。

### 対象ユーザー

- 中学生・高校生（主要ターゲット）
- 資格試験受験者
- 語学学習者
- 知識習得を効率化したいすべての学習者

---

## 🎯 主要機能

### 1. 適応型学習システム

ユーザーの学習状況に応じて、最適な出題タイミングと順序を自動調整します。

**特徴**:
- 6段階の記憶定着度管理（未学習 → 習熟済み）
- SM-2アルゴリズムベースの出題間隔調整
- 学習履歴に基づく個別最適化

### 2. クイズ機能

4択形式で単語を学習します。

**特徴**:
- CSV種類別の最適化された選択肢表示
- 回答後の詳細な解説表示
- 前へ/次へボタンで自由な移動
- 進捗の可視化（光るアニメーション）

### 3. 成績管理

詳細な学習履歴と統計を提供します。

**管理項目**:
- CSV別成績（ファイルごとの学習履歴）
- 単語別成績（個別単語の詳細統計）
- 時系列での成績推移
- 正答率、出題回数、連続正解数

### 4. CSV管理

柔軟な問題集管理機能を提供します。

**機能**:
- CSV一覧表示
- インポート・エクスポート
- アプリ内編集
- プレビュー機能

### 5. 学習モード

学習の目的に応じて3つのモードを選択できます。

- **通常モード**: バランス型の学習
- **復習モード**: 短期・中期記憶を重点復習
- **補習モード**: 苦手な単語を集中学習

## 🛠️ 技術スタック

### 開発環境

| 項目 | バージョン |
|-----|----------|
| **開発言語** | Swift 5.9+ |
| **フレームワーク** | SwiftUI |
| **IDE** | Xcode 15.0+ |
| **最小対応OS** | iOS 15.0 |
| **推奨OS** | iOS 17.0+ |

### アーキテクチャ

- **設計パターン**: Feature-First / Vertical Slice Architecture
- **状態管理**: SwiftUI @StateObject / @ObservedObject
- **データ永続化**: Core Data
- **非同期処理**: Swift Concurrency (async/await)

### 外部依存関係

現在、外部ライブラリへの依存はありません（Pure Swift実装）。

## 🎓 対応する学習コンテンツ

### CSV種類

プロジェクトは以下のCSV種類に対応しています：

1. **中学歴史**
   - ヘッダ: 年号、登場人物、史実名、解説、関連史実、関連分野、難易度
   - 特徴: 時系列学習に最適化

2. **中学古典単語**
   - ヘッダ: 語句、読み（ひらがな）、意味、語源等解説、関連語と意味、関連分野、難易度
   - 特徴: 古文学習に最適化

3. **中学英単語・英熟語**
   - ヘッダ: 語句、発音（カタカナ）、和訳、語源等解説、関連語と意味、関連分野、難易度
   - 特徴: 英語学習に最適化（発音表示機能付き）

4. **中学英会話**
   - ヘッダ: 語句、発音（カタカナ）、和訳、語源等解説、関連語と意味、関連分野、難易度
   - 特徴: 会話表現学習に最適化

5. **Xcode（プログラミング用語）**
   - ヘッダ: 語句、発音（カタカナ）、和訳、語源等解説、関連語と意味、関連分野、難易度
   - 特徴: 技術用語学習に対応
