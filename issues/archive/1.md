### Pythonで教学用の図形描画プログラムに関する検討

スマホアプリGitHubを使って相談して進行している内容を、このリポジトリにイシューとして記録します。

**概要**
- 東京書籍の中学数学指導計画に基づき、Pythonを用いて教学用問題として図形を生成するプログラム。
- 例：半径3cmの円の面積を求める問題の図形。
  - 問題で使用する図形描画プログラム例コード：
```python
import matplotlib.pyplot as plt

# 円を描画する関数
def draw_circle(radius):
    fig, ax = plt.subplots()
    circle = plt.Circle((0, 0), radius, color='blue', fill=False, linewidth=2)
    ax.add_artist(circle)
    ax.set_xlim(-radius - 1, radius + 1)
    ax.set_ylim(-radius - 1, radius + 1)
    ax.set_aspect('equal')  # 円を正しい比率で表示
    plt.title(f"半径 {radius}cm の円")
    ax.set_xlabel("X (cm)")
    ax.set_ylabel("Y (cm)")
    plt.grid(True)
    plt.show()

# 半径3cmの円を描画
draw_circle(radius=3)
```
